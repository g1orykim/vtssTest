monitor destination interface <port_type> <in_port_type>

 

Syntax Description

monitor

Set monitor configuration.

destination

The destination port. That is the port that trafficed should be mirrored to.

interface

Interface to mirror traffic to.

<port_type>

None.

<in_port_type>

None.

 

Command Mode

Global Configuration Mode

Privilege level

15
