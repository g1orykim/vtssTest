QoS Ingress Queue Policer Config Help

This page allows you to configure the Queue Policer settings for all switch ports.

The settings relate to the currently selected stack unit, as reflected by the page header.

The displayed settings are:

Port
The port number for which the configuration below applies.
Enabled (E)
Controls whether the queue policer is enabled on this switch port.
Rate
Controls the rate for the queue policer. The default value is ?. This value is restricted to ?-1000000 when the "Unit" is "kbps", and it is restricted to 1-? when the "Unit" is "Mbps".
This field is only shown if at least one of the queue policers are enabled.
Unit
Controls the unit of measure for the queue policer rate as kbps or Mbps. The default value is ?.
This field is only shown if at least one of the queue policers are enabled.

Buttons

: Click to save changes.

: Click to undo any changes made locally and revert to previously saved values.
