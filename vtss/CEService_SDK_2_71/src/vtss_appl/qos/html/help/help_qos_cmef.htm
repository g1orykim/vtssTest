Congestion Management Help

Configure Congestion Management on this page.

If Congestion Management is enabled then traffic is only forwarded to the stacking links if the destination port is uncongested.

Note that even when Congestion Management is enabled, it is not applied to priority 6 and 7, which are always allowed to be forwarded to the stacking links.

Mode
Indicates the Congestion Management mode. Possible modes are:
Enabled: Enable Congestion Management.
Disabled: Disable Congestion Management.

Buttons

: Click to save changes.

: Click to undo any changes made locally and revert to previously saved values.
