firmware swap

 

Syntax Description

firmware

Firmware upgrade/swap

swap

Swap between Active and Alternate firmware image.

 

Command Mode

User EXEC Mode

Privilege level

15

Syntax Description

firmware

upgrade

Firmware upgrade

<tftpserver_path_file>

Command Mode

15
