VLAN Membership Configuration Help

Delete
To delete a VLAN entry, check this box. The entry will be deleted on all stack switch units during the next Save.
VLAN ID
Indicates the ID of this particular VLAN.
Adding a New VLAN
Click to add a new VLAN ID. Legal values for a VLAN ID are 1 through 4095.

The VLAN is enabled on the selected switch unit when you click on "Save". A VLAN without any port members will be deleted when you click "Save".

The button can be used to undo the addition of new VLANs.

Buttons

: Click to save changes.

: Click to undo any changes made locally and revert to previously saved values.

: Click to go back to this MEP instance main page.

: Refreshes the displayed table starting from the "VLAN ID" input fields.
