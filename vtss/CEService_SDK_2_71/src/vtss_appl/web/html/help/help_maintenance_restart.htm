Restart Device

You can restart the stack switch on this page. After restart, the stack switch will boot normally.

Uncheck "Force Cool Restart" if you want to perform a warm restart of the stack. switch.
Check "Force Cool Restart" if you want to simulate a power-on (cold restart) of the stack. switch.
Warning: Forcing a cool restart will affect the traffic going through the stack. switch.

: Click to restart device.

: Click to return to the Port State page without restarting.
