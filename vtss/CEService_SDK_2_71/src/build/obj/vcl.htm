show vlan ip-subnet [ id <subnet_id> ]

 

Syntax Description

show

You can verify your settings by entering the show vlan protocol privileged EXEC command.

vlan

VLAN status

ip-subnet

Show VLAN ip-subnet entries.

id

Show a specific ip-subnet entry.

<subnet_id>

The specific ip-subnet to show.
