ping ipv6 <v_ipv6_addr> [ repeat <count> ] [ size <size> ] [ interval <seconds> ] [ interface vlan <v_vlan_id> ]

 

Syntax Description

ping

Send ICMP echo messages

ipv6

IPv6 (ICMPv6) echo

<v_ipv6_addr>

ICMPv6 destination address

repeat

Specify repeat count

<count>

1-60; Default is 5

size

Specify datagram size

<size>

2-1452; Default is 56 (excluding MAC, IP and ICMP headers)

interval

Specify repeat interval

<seconds>

0-30; Default is 0

interface

Select an interface to configure

vlan

VLAN Interface

<v_vlan_id>

VLAN identifier(s): VID

 

Command Mode

User EXEC Mode

Privilege level

0
