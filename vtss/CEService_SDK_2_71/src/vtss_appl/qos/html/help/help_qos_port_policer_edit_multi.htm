QoS Ingress Port Policer Config Help

This page allows you to configure the Policer settings for a specific port.

The settings relate to the currently selected stack unit, as reflected by the page header.

The displayed settings are:

Enabled
Controls whether the policer is enabled on this switch port.
Rate
Controls the rate for the policer. The default value is ?. This value is restricted to ?-1000000 when the "Rate Unit" is "kbps" or "fps", and it is restricted to 1-? when the "Rate Unit" is "Mbps" or "kfps".
Rate Unit
Controls the unit of measure for the policer rate as kbps, Mbps, fps or kfps . The default value is ?.
DP Bypass Level
Controls the Drop Precedence bypass level. Frames with Drop Precedence below the bypasss level are not policed.
Unicast
Controls whether unicast frames are policed.
Multicast
Controls whether multicast frames are policed.
Broadcast
Controls whether broadcast frames are policed.
Flooding
Controls whether flooded frames are policed.
Learning
Controls whether learning frames are policed.
Flow Control
If flow control is enabled and the port is in flow control mode, then pause frames are sent instead of discarding frames.

Buttons

: Click to save changes.

: Click to undo any changes made locally and revert to previously saved values.

: Click to undo any changes made locally and return to the overview.
