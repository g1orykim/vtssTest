green-ethernet led interval <v_0_to_24> intensity <v_0_to_100>

 

Syntax Description

green-ethernet

Green ethernet (Power reduction).

led

LED power reduction.

interval

Interval in whole hours at which to configure the LED intensity.

<v_0_to_24>

Interval from 00.00 to 24.00 (00 is used to start at midnight, while 24 is used to stop at midnight).

intensity

LEDs intensity.

<v_0_to_100>

Internsity from 0%% (LEDs OFF) to 100%%

 

Command Mode

Global Configuration Mode

Privilege level

15

 

 

green-ethernet led on-event { [ link-change <v_0_to_65535> ] [ error ] }

 

Syntax Description

green-ethernet

Green ethernet (Power reduction).

led

LED power reduction.

on-event

Specifies when to turn LEDs on at 100%% intensity.

link-change

Specifies how long to turn LEDs intensity to 100%%, when a link changes state.

<v_0_to_65535>

Number of seconds to set LEDs intensity at 100%% intensity at link change.

error

Set LEDs intensity to 100%% if an error occurs.

 

Command Mode

Global Configuration Mode

Privilege level

15
