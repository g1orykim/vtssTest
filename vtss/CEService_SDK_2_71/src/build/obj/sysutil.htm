password encrypted <encry_password>

 

Syntax Description

password

Specify the password for the administrator

encrypted

Specifies an ENCRYPTED password will follow

<encry_password>

The ENCRYPTED (hidden) user password. Notice the ENCRYPTED password will be decoded by system internally. You cannot directly use it as same as the Plain Text and it is not human-readable text normally.

 

Command Mode

Global Configuration Mode

Privilege level

15

 

 

password none

 

Syntax Description

password

Specify the password for the administrator

none

NULL password

 

Command Mode

Global Configuration Mode

Privilege level

15

 

 

password unencrypted <password>

 

Syntax Description

password

Specify the password for the administrator

unencrypted

Specifies an UNENCRYPTED password will follow

<password>

The UNENCRYPTED (Plain Text) user password. Any printable characters including space is accepted. Notice that you have no change to get the Plain Text password after this command. The system will always display the ENCRYPTED password.

 

Command Mode

Global Configuration Mode

Privilege level

15
