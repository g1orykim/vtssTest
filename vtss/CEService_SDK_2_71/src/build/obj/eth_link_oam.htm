clear link-oam statistics [ interface <port_type> [ <plist> ] ]

 

Syntax Description

clear

Clear

link-oam

Clear Link OAM statistics

statistics

Clear Rx/Tx counters

interface

Clear Link OAM statistic on a specific interface or all interfaces.

<port_type>

Port type in Fast, Giga or Tengiga ethernet

<plist>

List of Port ID, ex, 1/1,3-5;2/2-4,6

 

Command Mode

User EXEC Mode

Privilege level

15

link-oam link-monitor frame { [ window <error_window> ] [ threshold <error_threshold> ] }

 

Syntax Description

link-oam

Link OAM configuration

link-monitor

Configure link monitoring

frame

Configure frame error event thresholds and window for error frames that trigger an error-frame link event

window

Set the a window of time during which error frames are counted

<error_window>

Duration of the monitoring period in terms of seconds

threshold

Set a threshold in number of frames

<error_threshold>

Number of permissible errors frames in the period defined by error_window

 

Command Mode

Port List Interface Mode

Privilege level

15
