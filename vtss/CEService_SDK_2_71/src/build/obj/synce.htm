network-clock clk-source <clk_list> nominate { clk-in | { interface <port_type> <port> } }

 

Syntax Description

network-clock

network-clock

clk-source

clk-source - commands related to a specific clock source.

<clk_list>

Clock source number

nominate

Nominate a clk input to become a selectable clock source.

clk-in

Nominate the station clock input as a source. The PCB104 SyncE module supports 10 MHz station clock input

interface

Nominate an ethernet interface as a source

<port_type>

None.

<port>

None.

 

Command Mode

Global Configuration Mode

Privilege level

15

network-clock selector { { manual clk-source <v_uint> } | selected | nonrevertive | revertive | holdover | freerun }

 

Syntax Description

network-clock

network-clock

selector

Selection mode of nominated clock sources

manual

Selector is manually set to the chosen clock source

clk-source

clk-source - commands related to a specific clock source.

<v_uint>

Clock source number

selected

Selector is manually set to the pt. selected clocfk source (not possible in unlocked mode)

nonrevertive

Selector is automatically selecting the best clock source - non revertively

revertive

Selector is automatically selecting the best clock source - revertively

holdover

Selector is forced in holdover

freerun

Selector is forced in free run
