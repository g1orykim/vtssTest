show voice vlan [ oui <oui> | interface <port_type> [ <port_list> ] ]

 

Syntax Description

show

Show running system information

voice

Voice appliance attributes

vlan

Vlan for voice traffic

oui

OUI configuration

<oui>

OUI value

interface

Select an interface to configure

<port_type>

Port type in Fast, Giga or Tengiga ethernet

<port_list>

List of Port ID, ex, 1/1,3-5;2/2-4,6

 

Command Mode

User EXEC Mode

Privilege level

15
